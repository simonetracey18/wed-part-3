Project Overview
This website was built for *Pretty and Clawed*, a luxury nail salon. Part 3 focuses on adding JavaScript interactivity, improving forms, implementing a gallery lightbox, dynamic search, SEO best practices, and full form validation.

Features Added in Part 3
Functionality (JavaScript Enhancements)
- Interactive **Gallery Lightbox** – click any image to view it enlarged in a modal.
- **Dynamic Service Search/Filter** – real-time search on the services page.
- **Smooth Scrolling Navigation**.
- **Client-side Form Validation** with helpful error messages on enquiry and contact forms.
- Responsive mobile menu and enhanced user experience.

SEO Improvements
- Proper `<title>` tags and meta descriptions on all pages.
- Keyword optimisation and relevant meta keywords.
- Descriptive image `alt` texts and optimised file naming.
- Clean heading structure and semantic HTML.

 Forms
- Fully functional enquiry and contact forms with HTML5 + JavaScript validation.
- User-friendly feedback on submission.
